# Portal Berita

Website portofolio pribadi berbasis web yang dikembangkan untuk menampilkan profil, project, dan informasi kontak secara profesional dengan desain Gen Z modern.

## Fitur
- Halaman beranda dengan perkenalan singkat
- Navigasi responsif dengan burger menu di perangkat mobile
- Menampilkan daftar project secara otomatis dari GitHub
- Tampilan modern, clean, dan mobile-friendly
- Informasi kontak terintegrasi

## Teknologi
- HTML
- CSS
- JavaScript
- GitHub API
